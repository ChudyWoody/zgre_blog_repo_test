Konfiguracja dnsmasq

Plik dnsmasq.conf musisz edytować, aby ustawić serwer DHCP dla interfejsu Ethernet (enp0s31f6). Oto kroki:

Otwórz plik:

sudo nano /etc/dnsmasq.conf

Dodaj/zmodyfikuj następujące linie:

Interfejs, na którym działa serwer DHCP Link to heading

interface=enp0s31f6

Zakres adresów IP przydzielanych urządzeniom Link to heading

dhcp-range=192.168.2.10,192.168.2.100,12h

Adres bramy domyślnej (twój komputer, bo to on przekazuje internet) Link to heading

dhcp-option=3,192.168.2.1

Serwery DNS (możesz użyć Google lub innych) Link to heading

dhcp-option=6,8.8.8.8,8.8.4.4

Nie działaj jako serwer DNS na innych interfejsach Link to heading

bind-interfaces

Włącz i uruchom dnsmasq

Włącz usługę dnsmasq:

sudo systemctl enable dnsmasq sudo systemctl start dnsmasq
